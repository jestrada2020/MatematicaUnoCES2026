## Ejercicio (1d)


Demuestre las siguientes identidades:

+ ${\Large cot(x)+\dfrac{sen(x)}{1+cos(x)}=csc(x)}$
	   
+ ${\Large csc(x)+cot(x)=\dfrac{sen(x)}{1-cos(x)}}$

+ ${\Large sen(x)+cos(x)=\dfrac{sec(x)+csc(x)}{tan(x)+cot(x)}}$
	   
+ ${\Large tan(x)+cot(x)=sec(x)csc(x)}$

+ ${\Large tan(x)+cot(x)=cot(x)sec^{2}(x)}$

+ ${\Large \dfrac{1-sen(x)}{cos(x)}=\dfrac{cos(x)}{1+sen(x)}}$

+ ${\Large sen^{4}(x)-cos^{4}(x)=2sen^{2}(x)-1}$

+ ${\Large \dfrac{sec^{2}(x)(1+cos(x).tan(x))}{(tan(x)+sec(x))^{2}+1}=\dfrac{1}{2}}$

+ ${\Large \dfrac{sen(x)}{2\sqrt{1-cos(x)}}=\dfrac{1}{2}\sqrt{1+cos(x)}}$

+ ${\Large sen(x)(1+tan(x))+cos(x)(1+cot(x))=sec(x)+csc(x)}$

+ ${\Large \dfrac{csc(x)+cot(x)}{sen(x)+tan(x)}=csc(x)cot(x)}$
	   
+ ${\Large \dfrac{sen(x)-cos(x)+1}{sen(x)+cos(x)-1}=tan(x)+sec(x)}$	   
	   
+ ${\Large (sen(\frac{x}{2})+cos(\frac{x}{2}))^{2}=1+sen(x)}$	
	
+ ${\Large sen^{4}(x)=\dfrac{1}{8}(3-4cos(2x)+cos(4x))}$	
	
	
	
	
	
## Ejercicio (2d)

Demuestre las siguientes fórmulas trigonométricas:


Ángulo medio para la función tangente:

+ $tan^{2}\Big(\frac{x}{2}\Big)=\dfrac{1-cos(x)}{1+cos(x)}$
     
Ángulo doble para la función tangente:
     
+ $tan(2x)=\dfrac{2tan(x)}{1-tan^{2}(x)}$
     
Suma y Diferencia de ángulos.
     
+ $sen(a)sen(b)=\dfrac{1}{2}\Big(cos(a-b)-cos(a+b)\Big)$
     
+ $cos(a)cos(b)=\dfrac{1}{2}\Big(cos(a-b)+cos(a+b)\Big)$
     
+ $sen(b)cos(a)=\dfrac{1}{2}\Big(sen(a+b)-sen(a-b)\Big)$
